# Algebra-of-sets-using-C

|    OPERATION      |   OPERATOR  |
| :------------     | :---------: |
| Union             |      +      |
| Subtraction       |      -      |
| Intersection      |      *      |

### Example

Given sets  `A = {1, 3, 5}` and `B = {1, 2, 4}`, then:
```
------------------------------------------------
OPERATION STRING: A+B
------------------------------------------------
SOLUTION: {1, 3, 5, 2, 4}
------------------------------------------------
```
